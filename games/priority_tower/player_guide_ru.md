# Башня приоритета

## Раздел

**Списки и цели**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Башня обороняет дорожку от волн. Нужно выбрать самого опасного врага по формуле приоритета.

## Что видно в визуализации

Враги бегут к базе по линиям, у каждого видны скорость, дистанция и опасность. Башня поворачивается к выбранной цели.

## Что написать

Определите `choose_target(enemies)`. Верните `id` цели с максимальной оценкой.

## Входные данные

- `danger` - базовая опасность.
- `speed` - скорость.
- `distance` - расстояние до базы.
- Оценка в примерах: `danger * 3 + speed * 2 - distance`.

## Как думать

Это поиск максимума в списке. Для каждого врага посчитайте оценку и храните лучшую цель.

## Пример решения

```python
def choose_target(enemies):
    best = enemies[0]
    best_score = best["danger"] * 3 + best["speed"] * 2 - best["distance"]
    for enemy in enemies:
        score = enemy["danger"] * 3 + enemy["speed"] * 2 - enemy["distance"]
        if score > best_score:
            best = enemy
            best_score = score
    return best["id"]
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Выбрать ближайшего без учета опасности.
- Вернуть сам словарь врага.
- Не обновлять лучшую оценку вместе с целью.
