# Захват флага

Это соревновательная игра для 2-4 участников. У каждой команды один бот и один слот `player`.

На карте есть один общий флаг. Его нужно подобрать и принести на свою базу. Если игрок несет флаг, его можно сбить выстрелом: игрок замрет на несколько ходов, а флаг выпадет на карту.

## Контракт функции

Напишите функцию:

```python
def make_move(x, y, board, has_flag):
    ...
```

Она должна вернуть строку с действием.

Обычные ходы:

```python
"up"
"down"
"left"
"right"
"stay"
```

Выстрелы:

```python
"shoot_up"
"shoot_down"
"shoot_left"
"shoot_right"
```

Строительство стены рядом с собой:

```python
"build_up"
"build_down"
"build_left"
"build_right"
```

У каждого игрока ограниченный запас построенных стен. Построенные стены можно разрушать выстрелами.

## Координаты

Карта передается как список столбцов:

```python
cell = board[x][y]
```

`x` растет вправо, `y` растет вниз.

Перед чтением клетки проверяйте границы:

```python
0 <= x < len(board)
0 <= y < len(board[x])
```

## Значения в `board`

```python
-4  # соперник несет флаг
-3  # построенная стена, ее можно разрушить выстрелом
-2  # соперник без флага
-1  # каменная стена
 0  # пустая клетка
 1  # флаг лежит на земле
 2  # ваша база
 3  # чужая база
```

Если `has_flag == True`, значит флаг у вас. Тогда главная цель - найти клетку `2` и вернуться на свою базу.

Если `has_flag == False`, ищите клетку `1`. Если клетки `1` нет, значит флаг несет соперник. Такой соперник виден как `-4`.

## Правила

- Игроки ходят одновременно.
- Если несколько игроков пытаются попасть в одну свободную клетку, движок случайно выбирает одного игрока, который займет ее. Остальные остаются на местах.
- Если два игрока пытаются поменяться местами, они остаются на местах.
- Флаг подбирается, когда игрок приходит на клетку с флагом.
- Захват засчитывается, когда носитель флага приходит на свою базу.
- После захвата флаг возвращается в центр карты.
- Выстрел летит по прямой до стены, построенной стены, игрока или предела дальности.
- Если выстрел попал в носителя флага, флаг выпадает, а носитель замораживается.
- Построенная стена блокирует движение, но разрушается выстрелом.

## Простая стратегия

1. Если вы несете флаг, идите к своей базе (`2`).
2. Если видите носителя флага `-4` на одной линии без стен, стреляйте.
3. Если флаг лежит на земле, идите к клетке `1`.
4. Если флаг у соперника, попробуйте подойти к нему и ждать момента для выстрела.

## Пример

```python
DIRECTIONS = [
    ("right", 1, 0),
    ("down", 0, 1),
    ("left", -1, 0),
    ("up", 0, -1),
]


def make_move(x, y, board, has_flag):
    if has_flag:
        return step_to_value(x, y, board, 2)

    shot = shoot_carrier_if_visible(x, y, board)
    if shot:
        return shot

    if find_cells(board, 1):
        return step_to_value(x, y, board, 1)

    return step_next_to_carrier(x, y, board)


def shoot_carrier_if_visible(x, y, board):
    for action, dx, dy in DIRECTIONS:
        nx = x + dx
        ny = y + dy
        while 0 <= nx < len(board) and 0 <= ny < len(board[nx]):
            cell = board[nx][ny]
            if cell == -4:
                return "shoot_" + action
            if cell in (-3, -2, -1):
                break
            nx += dx
            ny += dy
    return None
```

Полный пример находится в `examples/demo.py`.

## Как усилить бота

- Стреляйте не только по носителю флага, но и по построенным стенам, если они перекрыли путь.
- Если вы несете флаг, иногда стройте стену за собой, чтобы замедлить преследователя.
- Не стреляйте, если между вами и целью стоит каменная стена.
- Проверяйте, не ведет ли ход в клетку рядом с соперником, который может сразу вас сбить.
- Сравнивайте несколько целей: флаг, носитель флага, своя база, удобная позиция для выстрела.
