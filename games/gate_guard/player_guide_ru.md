# Страж ворот

## Раздел

**Условия и выбор**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

У ворот появляются разные ситуации: красные ворота, синие ворота с ключом, ловушка или обычный проход. Нужно выбрать одно действие.

## Что видно в визуализации

Визуализация показывает героя перед воротами, цвет ворот, ловушку и подсказку о ключе. Правильный выбор открывает путь дальше.

## Что написать

Определите `choose_action(gate)`. Верните одну строку: `attack`, `use_key`, `disarm`, `open` или `wait`.

## Входные данные

- `gate["color"]` - цвет ворот.
- `gate["has_key"]` - есть ли ключ.
- `gate["trap"]` - есть ли ловушка.

## Как думать

Проверяйте условия от самых важных к обычным. Красные ворота требуют атаки, синие без ключа лучше не открывать, ловушку надо обезвредить.

## Пример решения

```python
def choose_action(gate):
    if gate["color"] == "red":
        return "attack"
    elif gate["color"] == "blue":
        if gate["has_key"]:
            return "use_key"
        return "wait"
    elif gate["trap"]:
        return "disarm"
    return "open"
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Проверить ловушку слишком поздно в ситуации, где она важна.
- Вернуть действие с опечаткой.
- Забыть ветку для обычных ворот.
