# Инвентарь героя

## Раздел

**Списки и цели**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Герой открывает рюкзак и выбирает предмет под ситуацию: лечение, ключ, факел или запасную еду.

## Что видно в визуализации

Вокруг героя показаны текущие условия, а предметы инвентаря лежат карточками внизу сцены.

## Что написать

Определите `choose_item(inventory, situation)`. Верните строку с названием предмета.

## Входные данные

- `inventory` - список строк.
- `situation["hp"]` - здоровье.
- `situation["door"]` - нужна ли дверь.
- `situation["dark"]` - темно ли вокруг.

## Как думать

Проверяйте ситуацию и наличие предмета одновременно: мало HP и есть зелье, дверь и есть ключ, темно и есть факел.

## Пример решения

```python
def choose_item(inventory, situation):
    if situation["hp"] < 40 and "healing_potion" in inventory:
        return "healing_potion"
    elif situation["door"] and "small_key" in inventory:
        return "small_key"
    elif situation["dark"] and "torch" in inventory:
        return "torch"
    return "apple"
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Выбрать предмет, которого нет в списке.
- Сравнивать весь список со строкой.
- Забыть запасной вариант.
