# Гравитационные яблоки

## Раздел

**Симуляции мира**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Яблоки падают вниз в своих столбцах, пока под ними пустая клетка. Нужно вернуть устойчивое поле.

## Что видно в визуализации

Сад показан сверху-сбоку: яблоки падают вниз по клеткам, камни блокируют падение.

## Что написать

Определите `simulate(board)`. Верните новую матрицу после падения яблок.

## Входные данные

- `A` - яблоко.
- `#` - камень.
- `.` - пустая клетка.
- Матрица хранится как `board[x][y]`, вниз означает увеличение `y`.

## Как думать

Удобно повторять проходы, пока хотя бы одно яблоко сдвинулось. Внутри столбца идите снизу вверх, чтобы яблоки не проходили сквозь друг друга.

## Пример решения

```python
def simulate(board):
    result = []
    for column in board:
        result.append(column[:])
    changed = True
    while changed:
        changed = False
        for x in range(len(result)):
            for y in range(len(result[x]) - 2, -1, -1):
                if result[x][y] == "A" and result[x][y + 1] == ".":
                    result[x][y] = "."
                    result[x][y + 1] = "A"
                    changed = True
    return result
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Идти сверху вниз и сдвигать яблоко несколько раз за один проход неожиданно.
- Изменить входной `board` и потерять исходные данные.
- Забыть остановить цикл, когда изменений больше нет.
