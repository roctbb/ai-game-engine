# Змейка

Напишите функцию `make_move(x, y, board)`, которая возвращает `up`, `down`, `left` или `right`.

- `x`, `y` — координаты головы змейки.
- `board` — матрица поля.

Значения в `board`:

- `0` — пустая клетка.
- `1` — еда.
- `-1` — препятствие: край поля или тело змейки.

```python
def make_move(x, y, board):
    if x + 1 < len(board[y]) and board[y][x + 1] != -1:
        return "right"
    return "down"
```

Цель: съесть как можно больше еды и остаться живым. Для отладки можно использовать `print(...)`.

Что важно:

- Нельзя развернуться на 180 градусов: если змейка идет вправо, ход `left` будет заменен на прежнее направление.
- Еда появляется в случайной свободной клетке.
- Хорошая стратегия сначала ищет еду в матрице, а потом выбирает безопасный соседний ход, который уменьшает расстояние до нее.
