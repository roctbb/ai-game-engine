# Пиксельный художник

## Раздел

**Матрицы и координаты**. Задача тренирует `pixels[x][y]`, вложенные циклы и создание новой матрицы.

## Цель игры

Есть матрица чисел. Каждое число - код цвета. Нужно вернуть такую же по размеру матрицу, но вместо чисел поставить названия цветов.

## Что видно в визуализации

На поле показана пиксельная картинка. Известные коды окрашиваются цветом, неизвестные коды отмечаются как прозрачные клетки.

## Входные данные

- `pixels[x][y]` - числовой код пикселя.
- `palette` - словарь, где ключ - код, значение - название цвета.

Если кода нет в `palette`, надо вернуть `"transparent"`.

## Как думать

Создайте пустой `result`. Для каждого столбца создайте `column`, пройдите по всем `y`, найдите цвет через `palette.get(code, "transparent")` и добавьте его в столбец.

## Пример решения

```python
def paint(pixels, palette):
    result = []
    for x in range(len(pixels)):
        column = []
        for y in range(len(pixels[x])):
            column.append(palette.get(pixels[x][y], "transparent"))
        result.append(column)
    return result
```

## Частые ошибки

- Перепутать `x` и `y`.
- Вернуть одну строку вместо матрицы.
- Использовать `palette[code]` и получить ошибку на неизвестном коде.
