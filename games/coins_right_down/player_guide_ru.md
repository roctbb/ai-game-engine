# Монеты вправо и вниз

Напишите функцию `make_move(state)`, которая возвращает `right` или `down`.

В `state` приходят:

- `position`: текущая позиция агента.
- `goal`: финишная клетка.
- `size`: размер поля.
- `coins`: оставшиеся монеты.
- `walls`: стены.
- `board`: матрица поля.
- `step`: номер текущего шага.

Значения в `board`:

- `-1` — стена.
- `0` — пустая клетка.
- `1` — монета.
- `2` — финиш.

Цель: дойти до финиша и собрать как можно больше монет. Карта каждый запуск генерируется случайно по `run_id`, но финиш всегда достижим, если аккуратно обходить стены.

Полезная стратегия: заранее оценивать лучший путь до финиша с учетом монет и стен.

```python
def make_move(state):
    x = state["position"]["x"]
    y = state["position"]["y"]
    board = state["board"]

    if x + 1 < len(board[y]) and board[y][x + 1] != -1:
        return "right"
    return "down"
```

Для отладки можно использовать `print(...)`. Вывод появится в консоли под визуализацией синхронно с кадрами replay.
