# Расшифровщик рун

## Раздел

**Строки и коды**. Руна - это символ строки, а строку удобно перебирать циклом `for`.

## Цель игры

Нужно перевести строку рун в список команд героя.

## Что видно в визуализации

На дорожке лежат руны. Над каждой руной показана команда, в которую она должна превратиться. Текущая руна подсвечивается.

## Что написать

```python
def decode(runes):
    ...
```

Верните список команд.

## Входные данные

`runes` - строка рун. Ее можно перебирать так же, как обычную строку.

## Таблица рун

- `"ᚠ"` -> `forward`
- `"ᚱ"` -> `turn_right`
- `"ᛚ"` -> `turn_left`
- `"ᚲ"` -> `collect`
- `"ᚨ"` -> `attack`
- `"᛫"` -> `wait`

## Как думать

Создайте словарь соответствий, затем пройдите по строке циклом `for`. На каждом шаге берите команду из словаря и добавляйте ее в ответ.

## Пример решения

```python
def decode(runes):
    mapping = {
        "ᚠ": "forward",
        "ᚱ": "turn_right",
        "ᛚ": "turn_left",
        "ᚲ": "collect",
        "ᚨ": "attack",
        "᛫": "wait",
    }

    result = []
    for rune in runes:
        result.append(mapping[rune])
    return result
```

## Частые ошибки

- Не добавить руну ожидания.
- Перепутать левый и правый поворот.
- Вернуть строку вместо списка команд.
