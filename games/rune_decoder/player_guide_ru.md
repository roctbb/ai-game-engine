# Расшифровщик рун

## Раздел

**Строки и коды**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Строка рун задает программу героя. Нужно перевести каждый символ в понятную команду.

## Что видно в визуализации

Руны лежат как светящиеся плитки, а герой выполняет расшифрованные действия на поле.

## Что написать

Определите `decode(runes)`. Верните список команд.

## Входные данные

- `ᚠ` - `forward`.
- `ᚱ` - `turn_right`.
- `ᛚ` - `turn_left`.
- `ᚲ` - `collect`.
- `ᚨ` - `attack`.

## Как думать

Сделайте словарь соответствий и пройдите циклом по строке. Каждый символ дает одну команду.

## Пример решения

```python
def decode(runes):
    mapping = {"ᚠ": "forward", "ᚱ": "turn_right", "ᛚ": "turn_left", "ᚲ": "collect", "ᚨ": "attack"}
    result = []
    for rune in runes:
        result.append(mapping[rune])
    return result
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Обрабатывать только первый символ.
- Вернуть строку команд вместо списка.
- Забыть одну из рун в словаре.

## Следующий шаг

После этой задачи попробуйте изменить пример: добавьте еще одну проверку, посчитайте промежуточную переменную или перепишите решение через более явный цикл. Так легче заметить, какая часть программы отвечает за выбор действия.
