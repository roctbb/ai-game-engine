# Классический сапер

Напишите функцию `choose_cell(board, flags_left)`.

Поле имеет размер 12×12 и 18 мин. Первая открытая клетка всегда безопасна, а карта подбирается так, чтобы ее можно было решить логическими правилами без угадывания.

`board` — матрица видимого поля:

- `-2` — закрытая клетка.
- `-1` — ваш флаг.
- `0..8` — открытая клетка с числом мин вокруг.

Функция должна вернуть:

- `(x, y)`, чтобы открыть клетку.
- `("flag", x, y)`, чтобы поставить или снять флаг.

Координаты считаются от левого верхнего угла: `x` вправо, `y` вниз.

```python
def choose_cell(board, flags_left):
    for y, row in enumerate(board):
        for x, cell in enumerate(row):
            if cell == -2:
                return (x, y)
    return (0, 0)
```

Цель: открыть все безопасные клетки и не попасть на мину. Для отладки можно использовать `print(...)`.

Полезные идеи для улучшения:

- Если число уже равно количеству флагов вокруг него, остальные закрытые соседи безопасны.
- Если число равно количеству флагов плюс закрытых соседей, все эти закрытые соседи можно пометить флагами.
- Начинайте с перебора соседей вокруг открытых чисел, а не со случайных клеток.
