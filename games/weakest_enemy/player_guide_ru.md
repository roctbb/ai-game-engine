# Арена слабейшего

## Раздел

**Списки и цели**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

На арене несколько врагов. Нужно выбрать врага с минимальным здоровьем, чтобы добить его первым.

## Что видно в визуализации

Враги стоят вокруг героя, над каждым показана полоска HP. Выбранная цель подсвечивается.

## Что написать

Определите `choose_target(enemies)`. Верните `id` врага с самым маленьким `hp`.

## Входные данные

- `enemies` - список словарей.
- У каждого врага есть `id` и `hp`.

## Как думать

Возьмите первого врага как лучший вариант, затем переберите остальных и заменяйте цель, если нашли меньший HP.

## Пример решения

```python
def choose_target(enemies):
    target = enemies[0]
    for enemy in enemies:
        if enemy["hp"] < target["hp"]:
            target = enemy
    return target["id"]
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Вернуть индекс вместо `id`.
- Искать максимум вместо минимума.
- Начать с пустого `target`, хотя список не пустой.
