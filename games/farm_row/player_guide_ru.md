# Грядка фермера

## Раздел

**Циклы и повторения**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Фермер идет вдоль одномерной грядки. Для каждой клетки нужно решить: поливать растение или пропустить пустую землю.

## Что видно в визуализации

На экране ряд клеток с ростками и пустыми участками. По мере проверки подсвечивается текущая клетка и команда ученика.

## Что написать

Определите `solve(row)`. Верните список команд той же длины, что и `row`: `water` для растения и `skip` для пустой клетки.

## Входные данные

- `row` - список чисел.
- `1` означает растение.
- `0` означает пустую клетку.

## Как думать

Здесь нужен цикл `for`: пройти по всем клеткам, для каждой добавить одну команду в список результата.

## Пример решения

```python
def solve(row):
    commands = []
    for cell in row:
        if cell == 1:
            commands.append("water")
        else:
            commands.append("skip")
    return commands
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Вернуть одну строку вместо списка.
- Добавить меньше команд, чем клеток.
- Перепутать `water` и `skip`.

## Следующий шаг

После этой задачи попробуйте изменить пример: добавьте еще одну проверку, посчитайте промежуточную переменную или перепишите решение через более явный цикл. Так легче заметить, какая часть программы отвечает за выбор действия.
