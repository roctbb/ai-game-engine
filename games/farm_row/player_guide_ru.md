# Грядка фермера

## Раздел

**Циклы и повторения**. Здесь нужен цикл по списку и простое условие внутри цикла.

## Цель игры

Фермер идет вдоль грядки. Нужно поливать только те клетки, где есть растение и влажность меньше 3.

## Что видно в визуализации

Каждая клетка показывает растение и шкалу влажности. Когда команда `water` выбрана правильно, появляется капля воды.

## Что написать

```python
def solve(row):
    ...
```

Верните список команд той же длины, что и `row`: `water` или `skip`.

## Входные данные

Каждая клетка - словарь:

```python
{"plant": True, "moisture": 2}
```

- `plant` показывает, есть ли растение.
- `moisture` показывает влажность почвы.

## Как думать

Идите по клеткам слева направо. Для каждой клетки отдельно решайте: если растения нет, пропускаем; если растение есть, сравниваем влажность с числом 3.

## Пример решения

```python
def solve(row):
    commands = []
    for cell in row:
        if cell["plant"] and cell["moisture"] < 3:
            commands.append("water")
        else:
            commands.append("skip")
    return commands
```

## Частые ошибки

- Поливать пустую клетку.
- Поливать растение, у которого влажность уже 3 или выше.
- Вернуть одну команду вместо списка команд.
