# Батарейный робот Lite

## Раздел

**Счетчики и ресурсы**. Здесь важен заряд батареи и безопасный запас до зарядной станции.

## Цель игры

Робот должен убирать грязь, двигаться вперед и вовремя возвращаться на зарядку. Если заряд почти закончился, уборка уже не главное.

## Что видно в визуализации

На поле показаны робот, зарядная станция, грязь, препятствие и батарейка робота. Если робот выбирает возврат, визуал подсвечивает путь к станции.

## Входные данные

`state` - словарь:

- `state["battery"]` - текущий заряд.
- `state["on_charger"]` - робот на зарядке.
- `state["dirty"]` - на текущей клетке грязь.
- `state["front_clear"]` - впереди свободно.
- `state["distance_to_charger"]` - сколько шагов до зарядки.

## Как думать

Сначала проверьте критический заряд на станции. Потом проверьте, хватит ли батареи на возвращение с запасом в 1 ход. Только после этого можно убирать грязь или двигаться.

## Пример решения

```python
def choose_action(state):
    if state["battery"] <= 2 and state["on_charger"]:
        return "charge"
    elif state["battery"] <= state["distance_to_charger"] + 1:
        return "return_to_charger"
    elif state["dirty"] and state["battery"] >= 2:
        return "clean"
    elif state["front_clear"] and state["battery"] >= 1:
        return "move"
    return "wait"
```

## Частые ошибки

- Чистить грязь при слишком низком заряде.
- Возвращаться на зарядку слишком поздно.
- Не проверять, свободна ли клетка впереди.
