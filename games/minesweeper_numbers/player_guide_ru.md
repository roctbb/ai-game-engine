# Сапер-редактор

## Раздел

**Матрицы и координаты**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Дана матрица мин. Нужно построить поле чисел: мина становится `-1`, пустая клетка получает количество мин вокруг.

## Что видно в визуализации

Поле сапера раскрывается клетка за клеткой, соседние мины подсвечиваются при подсчете.

## Что написать

Определите `build_numbers(mines)`. Верните матрицу чисел в формате `result[x][y]`.

## Входные данные

- `mines[x][y] == True` означает мину.
- Соседи - 8 клеток вокруг: по горизонтали, вертикали и диагоналям.
- Координаты за пределами поля проверять нельзя.

## Как думать

Для каждой клетки переберите `dx` и `dy` от `-1` до `1`. Пропустите пару `(0, 0)` и проверяйте границы перед обращением к матрице.

## Пример решения

```python
def build_numbers(mines):
    width = len(mines)
    height = len(mines[0])
    result = [[0 for y in range(height)] for x in range(width)]
    for x in range(width):
        for y in range(height):
            if mines[x][y]:
                result[x][y] = -1
            else:
                count = 0
                for dx in (-1, 0, 1):
                    for dy in (-1, 0, 1):
                        if dx == 0 and dy == 0:
                            continue
                        nx = x + dx
                        ny = y + dy
                        if 0 <= nx < width and 0 <= ny < height and mines[nx][ny]:
                            count += 1
                result[x][y] = count
    return result
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Считать саму клетку соседом.
- Выйти за границы матрицы.
- Вернуть `True/False` вместо чисел.
