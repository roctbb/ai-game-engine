# Сапер-редактор

## Раздел

**Матрицы и координаты**. Здесь не нужен BFS, графы или динамическое программирование. Главная цель - аккуратно работать с соседями клетки и границами поля.

## Цель игры

Дана матрица мин. Нужно построить поле чисел: мина становится `-1`, пустая клетка получает количество мин вокруг.

## Что видно в визуализации

Поле сапера раскрывается клетка за клеткой. Для активной клетки подсвечиваются все допустимые соседи. Это особенно важно на краях и в углах, где соседей меньше восьми.

## Что написать

Определите `build_numbers(mines)`. Верните матрицу чисел в формате `result[x][y]`.

## Входные данные

- `mines[x][y] == True` означает мину.
- Соседи - 8 клеток вокруг: по горизонтали, вертикали и диагоналям.
- Перед обращением к `mines[nx][ny]` нужно проверить, что координаты внутри поля.

## Как думать

Для каждой клетки переберите `dx` и `dy` от `-1` до `1`. Пропустите пару `(0, 0)`. Затем вычислите `nx` и `ny`, проверьте границы и только после этого смотрите, есть ли там мина.

## Пример решения

```python
def build_numbers(mines):
    width = len(mines)
    height = len(mines[0])
    result = [[0 for y in range(height)] for x in range(width)]

    for x in range(width):
        for y in range(height):
            if mines[x][y]:
                result[x][y] = -1
            else:
                count = 0
                for dx in (-1, 0, 1):
                    for dy in (-1, 0, 1):
                        if dx == 0 and dy == 0:
                            continue
                        nx = x + dx
                        ny = y + dy
                        if 0 <= nx < width and 0 <= ny < height and mines[nx][ny]:
                            count += 1
                result[x][y] = count

    return result
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, числа у краев и углов будут считаться без ошибок выхода за границы.

## Частые ошибки

- Считать саму клетку соседом.
- Обратиться к `mines[nx][ny]` до проверки границ.
- Вернуть `True/False` вместо чисел.
