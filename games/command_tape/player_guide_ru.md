# Командная лента

## Раздел

**Строки и коды**. Это маленький интерпретатор: каждый символ строки превращается в команду робота.

## Цель игры

Робот читает ленту слева направо и выполняет команды. Нужно вернуть список полных команд.

## Что видно в визуализации

Сверху показана лента символов. Текущий символ подсвечивается, робот двигается по полю, собирает предметы или ждет.

## Что написать

```python
def run_tape(tape):
    ...
```

## Входные данные

`tape` - строка команд. Каждый символ надо перевести в одну полную команду.

## Таблица команд

- `"F"` -> `forward`
- `"L"` -> `turn_left`
- `"R"` -> `turn_right`
- `"A"` -> `attack`
- `"C"` -> `collect`
- `"P"` -> `pickup`
- `"."` -> `wait`

## Как думать

Это похоже на переводчик. Сделайте словарь, где ключ - короткий символ, а значение - полное название команды, затем пройдите по строке в исходном порядке.

## Пример решения

```python
def run_tape(tape):
    mapping = {
        "F": "forward",
        "L": "turn_left",
        "R": "turn_right",
        "A": "attack",
        "C": "collect",
        "P": "pickup",
        ".": "wait",
    }

    commands = []
    for symbol in tape:
        commands.append(mapping[symbol])
    return commands
```

## Частые ошибки

- Забыть точку `.` как команду ожидания.
- Не добавить команду `P`.
- Изменить порядок команд при переводе.
