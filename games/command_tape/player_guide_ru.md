# Командная лента

## Раздел

**Строки и коды**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Робот читает строку вроде `FFRCA` и превращает ее в список действий.

## Что видно в визуализации

Командная лента движется над полем, текущий символ подсвечивается, а робот выполняет расшифрованное действие.

## Что написать

Определите `run_tape(tape)`. Верните список команд.

## Входные данные

- `F` - `forward`.
- `L` - `turn_left`.
- `R` - `turn_right`.
- `A` - `attack`.
- `C` - `collect`.

## Как думать

Это маленький интерпретатор. Сделайте словарь команд и добавляйте расшифровку каждого символа в результат.

## Пример решения

```python
def run_tape(tape):
    mapping = {"F": "forward", "L": "turn_left", "R": "turn_right", "A": "attack", "C": "collect"}
    commands = []
    for symbol in tape:
        commands.append(mapping[symbol])
    return commands
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Выполнить только первый символ.
- Вернуть исходную строку.
- Использовать названия команд с опечатками.
