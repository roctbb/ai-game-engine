# Ферма 2D

## Раздел

**Матрицы и координаты**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Фермеру нужно посчитать все растения на прямоугольной ферме.

## Что видно в визуализации

Камера сверху показывает ферму по клеткам. Сканер проходит по столбцам и строкам, найденные растения подсвечиваются.

## Что написать

Определите `count_plants(field)`. Верните одно число.

## Входные данные

- `field[x][y] == 1` - растение.
- `field[x][y] == 0` - пустая земля.
- Карта хранится как список столбцов: `x` потом `y`.

## Как думать

Нужны два вложенных цикла. Каждый раз, когда встречаете `1`, увеличивайте счетчик.

## Пример решения

```python
def count_plants(field):
    total = 0
    for x in range(len(field)):
        for y in range(len(field[x])):
            if field[x][y] == 1:
                total += 1
    return total
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Считать только первый столбец.
- Вернуть список вместо числа.
- Перепутать `len(field)` и `len(field[x])`.
