# Ферма 2D

## Раздел

**Матрицы и координаты**. Здесь не нужен BFS, графы или динамическое программирование. Главная цель - пройти по `field[x][y]` вложенными циклами и посчитать сумму.

## Цель игры

На ферме растут разные культуры. Нужно посчитать урожайность всей карты:

- `0` - пустая клетка;
- `1` - морковь, 1 очко;
- `2` - пшеница, 2 очка;
- `3` - кукуруза, 3 очка.

## Что видно в визуализации

Камера сверху показывает ферму по клеткам. Сканер проходит по столбцам и строкам, а на растениях видны очки урожая.

## Что написать

Определите `count_plants(field)`. Верните одно число - сумму всех значений в матрице.

## Входные данные

- `field[x][y]` - тип клетки.
- Карта хранится как список столбцов: сначала `x`, потом `y`.

## Как думать

Нужны два вложенных цикла. Для каждой клетки добавьте ее значение к общей сумме.

## Пример решения

```python
def count_plants(field):
    total = 0
    for x in range(len(field)):
        for y in range(len(field[x])):
            total += field[x][y]
    return total
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, итоговая урожайность совпадет с числом на поле.

## Частые ошибки

- Считать только количество непустых клеток, а не сумму очков.
- Считать только первый столбец.
- Перепутать `len(field)` и `len(field[x])`.

## Следующий шаг

Попробуйте отдельно посчитать урожайность каждого столбца и найти самый богатый.
