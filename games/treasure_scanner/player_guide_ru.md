# Сканер сокровищ

## Раздел

**Матрицы и координаты**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Сканер ищет самую богатую линию сокровищ: строку или столбец с максимальной суммой.

## Что видно в визуализации

Луч сканера проходит по строкам и столбцам, суммы показываются рядом с полем.

## Что написать

Определите `scan(board)`. Верните словарь вида `{"axis": "row" или "column", "index": номер, "score": сумма}`.

## Входные данные

- `board[x][y]` - число сокровищ в клетке.
- Сначала идет `x` - столбец, затем `y` - строка.
- Нужно сравнивать суммы линий, а не искать путь.

## Как думать

Посчитайте суммы всех строк и столбцов. Храните лучшую сумму и описание лучшей линии.

## Пример решения

```python
def scan(board):
    best = {"axis": "column", "index": 0, "score": sum(board[0])}
    for x in range(len(board)):
        total = sum(board[x])
        if total > best["score"]:
            best = {"axis": "column", "index": x, "score": total}
    height = len(board[0])
    for y in range(height):
        total = 0
        for x in range(len(board)):
            total += board[x][y]
        if total > best["score"]:
            best = {"axis": "row", "index": y, "score": total}
    return best
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Суммировать диагональ вместо строки/столбца.
- Перепутать ширину и высоту.
- Обновить сумму, но не обновить номер лучшей линии.
