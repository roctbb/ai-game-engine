# Выход из лабиринта

Напишите функцию `make_move(x, y, maze)`, которая на каждом шаге возвращает одно действие: `up`, `down`, `left` или `right`.

В функцию приходят:

- `x`: текущая координата по горизонтали.
- `y`: текущая координата по вертикали.
- `maze`: карта лабиринта как список списков чисел. Значения клеток: `-1` — стена, `0` — пустая клетка, `1` — выход.

Цель: дойти до выхода за меньшее число шагов и не пытаться ходить в стены. Коридоры узкие (1 клетка), но в лабиринте есть несколько путей к выходу — выбирайте кратчайший.
Карта 21×21, финиш выбирается случайно по достижимой клетке с длинным маршрутом от старта.
За скорость движения начисляется больше очков: чем ближе ваш путь к оптимальному, тем выше результат.
За некорректные действия (неизвестное действие / шаг в стену) даются штрафы.

Пример проверки клетки справа:

```python
def make_move(x, y, maze):
    if x + 1 < len(maze[y]) and maze[y][x + 1] != -1:
        return "right"
    return "down"
```

Для отладки можно использовать `print(...)`. Вывод появится в консоли под визуализацией синхронно с кадрами replay.
