# Кристаллы в коридоре

## Раздел

**Циклы и повторения**. Задача тренирует цикл `while`: идти, пока не встретилась стена.

## Цель игры

Герой идет по коридору до первой стены. Обычные и большие кристаллы надо собирать, проклятые кристаллы лучше пропустить.

## Что видно в визуализации

Коридор показан слева направо. Кристаллы отличаются цветом, стена останавливает героя, а текущая команда подсвечивается на поле.

## Что написать

```python
def solve(corridor):
    ...
```

Верните список команд `collect` и `move`.

## Входные данные

`corridor` - список клеток коридора слева направо.

## Клетки

- `"."` - пустая клетка.
- `"C"` - обычный кристалл, его надо собрать.
- `"B"` - большой кристалл, его тоже надо собрать.
- `"X"` - проклятый кристалл, его не собираем.
- `"#"` - стена, на ней цикл должен остановиться.

## Как думать

Используйте индекс и цикл `while`: пока индекс внутри списка и клетка не стена, обработайте клетку и сдвиньте индекс на 1.

## Пример решения

```python
def solve(corridor):
    commands = []
    index = 0

    while index < len(corridor) and corridor[index] != "#":
        if corridor[index] in ("C", "B"):
            commands.append("collect")
        commands.append("move")
        index += 1

    return commands
```

## Частые ошибки

- Продолжить идти после стены.
- Собрать проклятый кристалл `X`.
- Забыть команду `move` после сбора.
