# Кристаллы в коридоре

## Раздел

**Циклы и повторения**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Герой идет по коридору до первой стены. Если на клетке лежит кристалл, его нужно собрать, затем двигаться дальше.

## Что видно в визуализации

Коридор показан одной линией: герой, кристаллы, пустые клетки и стена. Анимация идет шаг за шагом до стены.

## Что написать

Определите `solve(corridor)`. Верните список команд `collect` и `move` до первой стены `#`.

## Входные данные

- `.` - пустая клетка.
- `C` - кристалл.
- `#` - стена, дальше идти нельзя.

## Как думать

Удобен цикл `while`: пока индекс внутри коридора и клетка не стена, обработайте клетку и сдвиньте индекс.

## Пример решения

```python
def solve(corridor):
    commands = []
    index = 0
    while index < len(corridor) and corridor[index] != "#":
        if corridor[index] == "C":
            commands.append("collect")
        commands.append("move")
        index += 1
    return commands
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Продолжить после стены.
- Забыть команду `move` после сбора кристалла.
- Не увеличить индекс в цикле.
