# Аукцион кристаллов

## Раздел

**Соревновательные стратегии**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Участники по очереди делают ставки на кристаллы. Нужно набрать больше ценности и не потратить бюджет слишком рано. Во второй половине игры кристаллы обычно дороже, поэтому весь бюджет в начале тратить опасно.

## Что видно в визуализации

В центре показан текущий кристалл, по сторонам бюджеты и накопленные очки участников. Победившая ставка забирает кристалл, а визуал показывает примерную честную ставку и выгоду сделки.

## Что написать

Для каждого слота определите `bid(state)`. Функция возвращает целое число - ставку на текущий кристалл.

## Входные данные

- `state["crystal_value"]` - ценность текущего кристалла.
- `state["budget"]` - ваш оставшийся бюджет.
- `state["rounds_left"]` - сколько раундов осталось.
- `state["my_value"]` - сколько ценности вы уже забрали на аукционе.
- `state["fair_bid"]` - примерная честная ставка для текущего кристалла.
- `state["average_value"]` - средняя ценность кристаллов в этом матче.

## Как думать

Ставка не должна быть больше бюджета. В начале полезно экономить, дорогие кристаллы можно брать смелее, а в последнем раунде нет смысла оставлять большой запас.

## Пример стратегии участника

```python
def bid(state):
    value = state["crystal_value"]
    budget = state["budget"]
    rounds_left = state["rounds_left"]
    if rounds_left <= 1:
        return min(budget, value)
    if value > state["average_value"]:
        return min(budget, state["fair_bid"] + 2)
    return min(budget, max(1, state["fair_bid"] - 2))
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Вернуть ставку больше бюджета.
- Всегда ставить весь бюджет в первом раунде.
- Игнорировать, что в конце могут прийти более дорогие кристаллы.
- Вернуть строку вместо числа.
