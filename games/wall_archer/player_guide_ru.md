# Лучник на стене

## Раздел

**Условия и выбор**. Задача учит принимать решение по нескольким числам: стрелы, здоровье, расстояние, урон и броня врага.

## Цель игры

Лучник защищает ворота. Нужно решить, стрелять, ставить щит, перезаряжаться или ждать.

## Что видно в визуализации

На стене стоит лучник, к воротам идет враг. Над врагом показаны здоровье, броня и угроза. Если выбран выстрел, летит стрела; если щит - появляется защитный круг.

## Что написать

```python
def choose_action(enemy, arrows, hp):
    ...
```

Верните `shoot`, `reload`, `shield` или `wait`.

## Входные данные

- `enemy["distance"]` - расстояние до врага.
- `enemy["hp"]` - здоровье врага.
- `enemy["damage"]` - урон врага.
- `enemy["armor"]` - броня врага.
- `enemy["is_aiming"]` - целится ли враг.
- `arrows` - сколько стрел осталось.
- `hp` - здоровье лучника.

## Правило выбора

Сначала посчитайте угрозу:

```python
threat = enemy["damage"] - enemy["distance"] * 2
```

Потом выберите действие:

1. Если стрел нет - `reload`.
2. Если враг целится, угроза не меньше 20 и здоровье меньше 50 - `shield`.
3. Если враг близко и броня не больше 1 - `shoot`.
4. Если у врага мало здоровья и броня не больше 1 - `shoot`.
5. Иначе - `wait`.

## Как думать

Не стреляйте сразу по любому врагу. Сначала проверьте стрелы, затем опасность для лучника, и только потом решайте, стоит ли тратить стрелу.

## Пример решения

```python
def choose_action(enemy, arrows, hp):
    threat = enemy["damage"] - enemy["distance"] * 2

    if arrows <= 0:
        return "reload"
    elif enemy["is_aiming"] and threat >= 20 and hp < 50:
        return "shield"
    elif enemy["distance"] <= 3 and enemy["armor"] <= 1:
        return "shoot"
    elif enemy["hp"] <= 25 and enemy["armor"] <= 1:
        return "shoot"
    return "wait"
```

## Частые ошибки

- Стрелять без стрел вместо перезарядки.
- Игнорировать броню врага.
- Ставить щит против слабой угрозы и терять ход.
