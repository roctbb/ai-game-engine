# Защита базы башнями

Напишите функцию `place_tower(state)`, которая возвращает номер линии для постройки башни: `0`, `1` или `2`.

В `state` приходят:

- `tick`: номер текущего хода.
- `lanes`: число линий.
- `track_length`: длина пути врагов.
- `base_hp`: здоровье базы.
- `budget`: доступный бюджет.
- `towers`: сколько башен стоит на каждой линии.
- `enemies`: список врагов с линией, позицией и здоровьем.

Цель: сохранить базу и уничтожить как можно больше врагов. Башня стоит 5 единиц бюджета; если бюджета не хватает, действие не строит башню.

Для отладки можно использовать `print(...)`. Вывод появится в консоли под визуализацией синхронно с кадрами replay.
