# TanksAI Classic

Это адаптация старых танчиков из `TanksAI` для платформы. Существующие танки платформы находятся в другом пакете и не используются здесь.

## Цель

Матч идет до 300 ходов или пока на арене не останется живых танков. Побеждает танк с наибольшим числом очков.

Очки считаются так:

```python
монеты * 50 + попадания * 20 + время_жизни - ошибки * 5
```

В начале раунда каждый танк получает 50 здоровья и 25 щита. На карте сразу появляется 30 монет, затем новые монеты могут появляться по ходу игры.

## Что написать

В слоте `bot` определите функцию:

```python
def make_choice(x, y, field):
    return "go_right"
```

Функция получает координаты вашего танка и карту `field`. Верните одну строку:

- `go_up`, `go_down`, `go_left`, `go_right` - движение на одну клетку.
- `fire_up`, `fire_down`, `fire_left`, `fire_right` - выстрел по прямой.
- `shield` - включить щит на этот ход.
- `self_destruct` - самоуничтожиться и нанести 3 урона танкам в радиусе 3 клеток, если взрыв не перекрыт стеной.

Выстрел летит до первого танка или стены. Монета тоже останавливает луч, как в старой визуализации.

## Формат карты

Карта - список столбцов: сначала `x`, затем `y`.

```python
width = len(field)
height = len(field[0])
cell = field[x][y]
```

Значения клетки:

- `0` - пусто.
- `1` - монета.
- `-1` - стена.
- `dict` - танк.

Словарь танка:

```python
{
    "name": "Игрок",
    "life": 50,
    "shield": 25,
    "score": 120,
    "history": ["go_right", "fire_up"]
}
```

## Пример

```python
def make_choice(x, y, field):
    width = len(field)

    if x + 1 < width and field[x + 1][y] in (0, 1):
        return "go_right"

    return "fire_right"
```

Для поиска монет удобен BFS. Готовый пример есть в `examples/miner_bfs.py`.
