# Яблочный рынок

Это соревновательная игра для 2-4 команд. В каждой команде ровно одна роль: `player`.

Движок сам расставляет команды по углам карты. Если команд две, они стартуют в противоположных углах. Если команд четыре, все четыре угла заняты.

## Цель

Собирайте яблоки на карте и относите их на свою базу. Чем больше яблок доставлено за 150 ходов, тем выше счет.

Игрок может нести максимум 2 яблока. Если рюкзак заполнен, обычно выгодно идти к базе.

В центре карты стоит яблоня. Через нее нельзя пройти, зато каждые 6 ходов рядом с ней падают новые яблоки. Поэтому матч не заканчивается после первых яблок: нужно постоянно выбирать, когда идти к дереву, когда возвращаться на базу и когда мешать соперникам броском.

## Контракт функции

Напишите функцию:

```python
def make_move(x, y, board, carrying):
    ...
```

Параметры:

- `x`, `y` - текущая позиция вашего игрока.
- `board` - карта как список столбцов: первая координата `x`, вторая `y`.
- `carrying` - сколько яблок вы сейчас несете: `0`, `1` или `2`.

Клетку читают так:

```python
cell = board[x][y]
```

`x` растет вправо, `y` растет вниз.

## Значения на карте

- `-1` - стена.
- `-2` - любой соперник.
- `0` - пустая клетка.
- `1` - яблоко.
- `2` - ваша база.
- `3` - яблоня в центре карты. Это препятствие, но рядом с ним появляются новые яблоки.

Каждая команда видит свою базу как `2`. Базы соперников не нужно искать отдельно.

## Действия

Функция должна вернуть одну строку:

- `"up"` - шаг вверх.
- `"down"` - шаг вниз.
- `"left"` - шаг влево.
- `"right"` - шаг вправо.
- `"stay"` - стоять на месте.
- `"throw_up"` - бросить яблоко вверх.
- `"throw_down"` - бросить яблоко вниз.
- `"throw_left"` - бросить яблоко влево.
- `"throw_right"` - бросить яблоко вправо.

Бросок работает только если `carrying > 0`. Один бросок тратит одно яблоко. Яблоко летит по прямой до 5 клеток, останавливается об стену и замораживает первого соперника на пути на несколько ходов.

Замороженный игрок пропускает ходы, поэтому бросок полезен, когда соперник стоит на одной линии с вами.

## Первый план решения

1. Если рядом по прямой есть соперник и вы несете яблоко, бросьте в него.
2. Если `carrying == 2`, найдите путь к базе `2`.
3. Иначе найдите путь к ближайшему яблоку `1`.
4. Если яблок сейчас не видно, можно двигаться ближе к центру или к базе: скоро рядом с яблоней появятся новые яблоки.
5. Если цели нет, сделайте любой безопасный ход или верните `"stay"`.

Карта и яблоки генерируются случайно, поэтому нельзя заранее записать готовую последовательность ходов.

## Простой пример

Этот пример не ищет лучший путь. Он показывает формат функции и не является готовым решением для победы.

```python
DIRECTIONS = {
    "up": (0, -1),
    "right": (1, 0),
    "down": (0, 1),
    "left": (-1, 0),
}


def in_bounds(x, y, board):
    return 0 <= x < len(board) and 0 <= y < len(board[x])


def can_step(x, y, board):
    return in_bounds(x, y, board) and board[x][y] not in (-1, 3)


def try_throw(x, y, board, carrying):
    if carrying <= 0:
        return None

    for action, (dx, dy) in DIRECTIONS.items():
        tx = x
        ty = y
        for _ in range(5):
            tx += dx
            ty += dy
            if not in_bounds(tx, ty, board) or board[tx][ty] in (-1, 3):
                break
            if board[tx][ty] == -2:
                return "throw_" + action
    return None


def make_move(x, y, board, carrying):
    throw_action = try_throw(x, y, board, carrying)
    if throw_action:
        return throw_action

    for action, (dx, dy) in DIRECTIONS.items():
        nx = x + dx
        ny = y + dy
        if can_step(nx, ny, board):
            return action

    return "stay"
```

## Идея сильнее: BFS

Для хорошей стратегии удобно искать кратчайший путь:

- если несете 2 яблока, цель - база `2`;
- иначе цель - ближайшее яблоко `1`;
- соперников `-2` лучше считать занятыми клетками;
- стены `-1` и яблоня `3` непроходимы.

Пример с BFS есть в папке `examples`.

## Где смотреть примеры

- `examples/demo.py` - базовая стратегия для одной роли `player`.
- `examples/north_bfs.py` - BFS с бросками.
- `examples/south_bfs.py` - BFS с другим порядком обхода и бросками.

Даже если файл называется `north_bfs.py` или `south_bfs.py`, в новой версии это просто разные демо-стратегии для одного слота `player`.

## Как начисляются очки

Главное - доставленные яблоки. Также есть небольшой бонус за удачные броски и штраф за неправильные ходы.

Полезные данные в результате матча:

- `scores` - счет команд.
- `winner_slots` - победившие внутренние позиции.
- `placements` - места команд.
- `metrics.delivered` - сколько яблок доставлено каждым игроком.
- `metrics.turn_limit` - лимит ходов матча.
- `metrics.apples_spawned_total` - сколько яблок всего появилось за матч.
- `metrics.throws` - сколько бросков сделал каждый игрок.
- `metrics.frozen` - сколько ходов заморозки осталось к концу матча.

## Частые ошибки

- Перепутать координаты и написать `board[y][x]` вместо `board[x][y]`.
- Идти в стену `-1`.
- Идти в яблоню `3`: это тоже препятствие.
- Бросать яблоко, когда `carrying == 0`.
- Искать только одного конкретного соперника: на карте любой соперник отмечен как `-2`.
- Забыть вернуться на базу, когда уже несете 2 яблока.
