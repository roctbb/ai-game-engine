# Зельевар

## Раздел

**Счетчики и ресурсы**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

В лаборатории есть ресурсы и рецепты. Нужно выбрать самое ценное зелье, на которое хватает ингредиентов.

## Что видно в визуализации

На столе видны ингредиенты, котлы с рецептами и выбранное зелье. Подсветка показывает, хватает ли ресурсов.

## Что написать

Определите `choose_potion(resources, recipes)`. Верните имя лучшего зелья.

## Входные данные

- `resources` - словарь доступных ингредиентов.
- `recipes` - словарь рецептов.
- В каждом рецепте есть ингредиенты и поле `value` - ценность зелья.

## Как думать

Переберите рецепты. Для каждого проверьте, хватает ли всех ингредиентов, а затем сравните `value` с лучшим найденным.

## Пример решения

```python
def choose_potion(resources, recipes):
    best_name = None
    best_value = -1
    for name, recipe in recipes.items():
        can_make = True
        for resource, amount in recipe.items():
            if resource == "value":
                continue
            if resources.get(resource, 0) < amount:
                can_make = False
        if can_make and recipe["value"] > best_value:
            best_name = name
            best_value = recipe["value"]
    return best_name
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Сравнить цену до проверки ингредиентов.
- Считать `value` ингредиентом.
- Вернуть весь рецепт вместо имени зелья.
