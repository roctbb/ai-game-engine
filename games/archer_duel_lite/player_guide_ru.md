# Дуэль лучников Lite

## Раздел

**Соревновательные стратегии**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Два участника управляют лучниками. Каждый ход стратегия выбирает: прицелиться, выстрелить, поднять щит или перезарядиться.

## Что видно в визуализации

На поле две башни, лучники, летящие стрелы, щиты, HP и количество стрел. Видно, когда участник целится и когда защищается.

## Что написать

Для каждого слота определите `make_move(state)`. Функция возвращает `aim`, `shoot`, `shield` или `reload`.

## Входные данные

- `state["hp"]` - здоровье вашего лучника.
- `state["arrows"]` - стрелы.
- `state["enemy_hp"]` - здоровье соперника.
- `state["enemy_aimed"]` - целится ли соперник.

## Как думать

Это не поиск пути, а стратегия по состоянию. Без стрел нужно перезаряжаться, при угрозе и низком HP полезен щит, а слабого соперника можно добивать выстрелом.

## Пример стратегии участника

```python
def make_move(state):
    if state["arrows"] <= 0:
        return "reload"
    if state["enemy_aimed"] and state["hp"] < 60:
        return "shield"
    if state["enemy_hp"] <= 35:
        return "shoot"
    return "aim"
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Стрелять без стрел.
- Всегда ставить щит и не выигрывать по урону.
- Вернуть `attack` вместо `shoot`.
