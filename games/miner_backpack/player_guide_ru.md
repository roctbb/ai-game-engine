# Рюкзак шахтера

## Раздел

**Счетчики и ресурсы**. Задача тренирует переменную-счетчик: сколько веса уже лежит в рюкзаке.

## Цель игры

Шахтер идет по тоннелю и видит куски руды. У каждого куска есть вес и ценность. Нужно вернуть команду `take` или `skip` для каждого куска.

## Что видно в визуализации

На поле показаны куски руды, их вес и ценность. Рядом с героем виден прогресс рюкзака: сколько веса уже занято из максимальной вместимости.

## Входные данные

- `ores` - список кусков руды.
- Каждый кусок выглядит так: `{"weight": 2, "value": 5}`.
- `capacity` - максимальный вес рюкзака.

## Как думать

Идите по списку слева направо. Храните `total_weight`. Кусок можно взять, если он помещается и его ценность не меньше веса. Если не подходит, верните `skip` и идите дальше.

## Пример решения

```python
def solve(ores, capacity):
    total_weight = 0
    commands = []

    for ore in ores:
        weight = ore["weight"]
        value = ore["value"]

        if total_weight + weight <= capacity and value >= weight:
            commands.append("take")
            total_weight += weight
        else:
            commands.append("skip")

    return commands
```

## Частые ошибки

- Увеличивать вес рюкзака даже для команды `skip`.
- Проверять только вместимость и забыть про ценность.
- Вернуть число вместо списка команд.
