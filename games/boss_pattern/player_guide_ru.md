# Маг против босса

## Раздел

**Режимы поведения**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Босс действует по повторяющемуся циклу. Маг должен выбрать правильную реакцию на состояние босса.

## Что видно в визуализации

На арене видны маг, босс, шкалы здоровья и текущая фаза атаки. Сильная атака подсвечивается заранее.

## Что написать

Определите `choose_action(turn, boss_state)`. Верните `shield`, `attack`, `area_spell` или `heal`.

## Входные данные

- `turn` - номер хода.
- `boss_state` - текущее действие босса, например `heavy_attack` или `summon`.

## Как думать

Сначала реагируйте на опасные состояния: от сильной атаки нужен щит, призыв лучше закрыть массовым заклинанием. В спокойный ход можно атаковать.

## Пример решения

```python
def choose_action(turn, boss_state):
    if boss_state == "heavy_attack":
        return "shield"
    elif boss_state == "summon":
        return "area_spell"
    return "attack"
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Игнорировать `boss_state` и смотреть только на номер хода.
- Вернуть действие с другим названием.
- Лечиться каждый ход и не наносить урон.
