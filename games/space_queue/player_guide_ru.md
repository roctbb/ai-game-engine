# Очередь кораблей

## Раздел

**Списки и цели**. Эта задача стоит в начале курса: здесь не нужен BFS, графы или динамическое программирование. Главная цель - потренировать базовый прием Python на понятной игровой сцене.

## Цель игры

Диспетчер космопорта должен упорядочить корабли: важные обслуживаются раньше, а среди равных раньше идет корабль с меньшим топливом.

## Что видно в визуализации

Корабли ждут у доков, у каждого показан приоритет и запас топлива. Очередь строится слева направо.

## Что написать

Определите `schedule(ships)`. Верните список `id` кораблей в порядке обслуживания.

## Входные данные

- `ships` - список словарей.
- `priority` - чем больше, тем важнее.
- `fuel` - чем меньше, тем срочнее при равном приоритете.

## Как думать

Можно использовать `sorted` с ключом: сначала отрицательный приоритет, затем топливо.

## Пример решения

```python
def schedule(ships):
    ordered = sorted(ships, key=lambda ship: (-ship["priority"], ship["fuel"]))
    result = []
    for ship in ordered:
        result.append(ship["id"])
    return result
```

## Как проверить себя

Запустите пример и посмотрите на анимацию. Если решение правильное, действия на поле совпадают с ожидаемыми шагами, счет доходит до максимума, а финальный кадр показывает победу или успешный выбор.

## Частые ошибки

- Отсортировать приоритет по возрастанию.
- Вернуть список кораблей вместо списка `id`.
- Потерять корабль при ручной сортировке.
